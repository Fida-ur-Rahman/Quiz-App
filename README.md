# Quiz App

A responsive **Quiz App** web application built using **HTML, CSS, and JavaScript**. This project allows users to take multiple-choice quizzes, receive instant feedback, and view their final score.

##  Features

- Multiple-choice questions
- Instant feedback for correct or incorrect answers
- Displays final score at the end of the quiz
- Responsive and mobile-friendly design
- Easy to add or modify quiz questions

##  Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

##  How It Works

1. Users start the quiz by clicking the start button.
2. Questions are displayed one at a time with multiple choices.
3. JavaScript tracks the user’s answers and provides instant feedback.
4. At the end, the total score is displayed with a summary.

##  How to Use

1. Clone the repository:
   ```
   git clone https://github.com/Fida-ur-Rahman/Quiz-App.git
   ```
2. Open the project folder.
3. Run `index.html` in your browser.
4. Start the quiz, select answers, and see your score at the end.
